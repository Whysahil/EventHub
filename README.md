# 🎟️ EventHub

EventHub is a full-stack event discovery and booking platform built for exploring and managing events through a modern and responsive web experience.

## Features

* User Registration & Login
* JWT Authentication
* Google OAuth Login
* GitHub OAuth Login
* Event Browsing
* Event Details Page
* Ticket Booking System
* My Tickets Management
* Responsive Design
* PostgreSQL Database Integration

## Tech Stack

### Frontend

* React
* TypeScript

### Backend

* Node.js
